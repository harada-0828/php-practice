# php-control-practice

## 概要
COACHTECH 教材 Tutorial 7-2「制御構文 ハンズオン演習」で作成した成果物です。
個人の得点からA〜Fの評価を行う成績判定システムです。合格・不合格者数また平均点の統計情報も算出します。

## 使用技術
PHP 8.x
docker(開発環境)


## 学んだこと
 関数を使ったコーディングの作業手順、例に基づいて条件分岐の使い方について学んだ。また、これらのコードを覚えることも重要だが、必要な場面で思いつく経験が必要なことを学びました。
今回のハンズオンではコーディングが複雑でAIの有用性を特に感じました。


## 動作確認
Docker環境を起動（docker compose up -d）し、ローカルサーバー経由でChromeブラウザから該当のURLにアクセスして、目的通りの画面表示がされているか確認を行った。
