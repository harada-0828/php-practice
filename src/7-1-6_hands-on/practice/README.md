# php-basics-practice

## 概要

COACHTECH 教材 Tutorial 7-1「PHPの基礎 ハンズオン演習」で作成した成果物です。
ノートパソコンの商品ページを作成しました。記載内容は商品の単価・在庫数。
また、在庫分の合計金額（小計）と税額（消費税）を合わせた（合計金額）を計算するプログラムです。

## 使用技術

PHP 8.x　
Docker （開発環境）

## 学んだこと

・PHPの基本構成やechoと各記号（" ; .）を使った構文の作成方法について。
・変数の基本構成と代入の方法について。
・phpハンズオンでは、基礎を学ぶための前提条件を元にphpのプログラム構成を学んだが、画面に表示する　方法は多様でありベストな方法を選ぶ為にも様々な知識を覚えておく方が良いプログラムを作ることが出来　ることを学びました。

## 動作確認

作成したPHPファイルをdockerで作成したローカルサーバー上で開き、PHPファイルの内容が予定通りに画面表示されるかの確認を行った。（以下手順）

１、ターミナルにて　docker compose up -d を実行し、ローカルサーバーを起動。
２、ブラウザで下記のURLにアクセス
　　http://localhost:8000/7-1-6_hands-on/practice/price_calculator.php
３、成果物の画面上での表示・価格計算が予定通りに実行されているか確認。
