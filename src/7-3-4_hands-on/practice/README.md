# php-oop-practice

## 概要

COACHTECH 教材 Tutorial 7-3「オブジェクト指向 ハンズオン演習」で作成した成果物です。
ユーザーの名前と年齢を追加するだけで自己紹介の文章を作成するプログラムです。また、年齢から成人か未成年化の判定を行うプログラムです。

## 使用技術

PHP 8.x
docker（開発環境）

## 学んだこと

オブジェクト指向の考え方・構成の為のプロパティとメソッドの定義を学んだ。また、変数・関数だけではできない大量のデータを効率よく処理する配列やループ（foreach）など目的に沿った扱い方を学んだ。

## 動作確認

docker環境下でローカルサーバーと、Chromeブラウザーを使い、指定のパスを入力した上で期待通りの画面表示が行われているかの確認を行った。その際に変数の内容や追加などを行いプログラムがきちんと働いているかの作業も行った。
